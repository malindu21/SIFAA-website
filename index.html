Introduction

Technology has become a vital aspect of functioning the world, for various functional purposes throughout the world. while And some industries like floriculture have not adapted technology to get satisfaction to solve and facilitate currently facing problems and provide the supply to the demand. Consequently, we have identified and implemented a solution that will address major aspects of this industry barrier. To address these major aspects we proposed a system Smart Intelligent Floriculture Assistant Agent (SIFAA), which uses expert knowledge with solutions and guidelines such as identifying diseases based on deep learning techniques. It also suggests remedies for diseases based on expert knowledge, recommends the best products for customers by using the Reinforcement Learning (RL) technique, motivate cultivators by using demand forecasting, and apply feature engineering by using Linear Regression (LR) and ensemble advanced LightGBM Regressors techniques.

We have conducted a survey to get feedback from floriculturists to find out the importance of proposed IT based solution to identified problems.

Methodology

The research outcome is an intelligent assistant which facilitates the floriculture industry where the core requirements are cultivating and selling harvested products. The SIFAA was developed to answer farmers’ cultivation queries, flower diseases identification, and predict the sales and demand. Furthermore, the system supports personalized behavior with the floriculture products recommendation. To facilitate these functionalities system comprises three components as follows,

  1. Floriculture disease detection using DCNN.
  2. Question and Answering as a human-counterpart.
  3. Automated floriculture product recommendation with demand forecasting.
  4. Using specific optimize algorithm to reach optimization the system.

Android application was built to facilitate implemented functionalities to end-user. Flask framework used as a bridge between the mobile application and above-mentioned functionalities Fig.1 shows the system architecture.

Integrated System

We have integrated our system using Android mobile application and connected our models through Flask RESTful API.

Conclusion

In this research, a proposed SIFAA was presented for helping farmers in the floriculture field. System provides most appropriate solution for diseases along with answers for user’s questions related to the floriculture by using acquired tacit knowledge of an expert and is capable of recommending most demanding flowers. Furthermore, the availability of diagnosing flower specific diseases has been maximized, since there is no human intervention enabling to gain a higher level of accuracy.
